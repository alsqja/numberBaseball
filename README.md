# 숫자 야구 게임

주제 : 자바를 이용한 숫자 야구 게임 만들기

# 계산기

주제 : 자바를 이용한 계산기 만들기

## 기능
- **Essential**
  - 정답 숫자 생성
  - 게임시작, 기록보기, 종료하기 입력 후 맞는 역할 수행
  - 정답 숫자 입력
  - 정답이 입력될 때 까지 계속 입력 가능
  - 정답을 맞추면 역할 메뉴 다시 선택
  - 정답 입력, 역할 입력 예외 처리
- **Advanced**
  - Essential 모든 기능
  - 게임 결과(맞추기 까지 걸린 횟수)들을 저장
  - 3-5 자리 난이도 설정
  - 난이도 입력 예외 처리
  - 난이도 입력 시 역할 메뉴 없이 바로 게임 시작

## Requirements
- **Essential**
  - [x]  **정답 숫자 생성하기**
    - [x]  정답은 서로 다른 3자리 수이다.
    - [x]  각 자리는 1~9 사이의 숫자이다. 0은 사용할 수 없습니다.
    - [x]  동일한 숫자는 사용될 수 없다. 즉, 숫자는 중복되지 않아야 한다.
  - [x]  **정답을 맞추기 위해 숫자를 입력하기**
    - [x]  서로 다른 3자리 수를 입력할 수 있다.
    - [x]  동일한 숫자는 사용될 수 없다. 즉, 숫자는 중복되지 않아야 한다.
    - [x]  숫자만 입력 가능하며, 문자는 작성할 수 없다.
  - [x]  **결과값 출력 및 게임 로직 적용하기**
    - [x]  정답과 입력값을 비교해 힌트를 “볼, 스트라이크, 아웃”으로 표시합니다.
        - [x]  스트라이크 : 입력값과 정답을 비교해 같은 자리에 같은 숫자가 있는 경우
        - [x]  볼 : 숫자는 같지만, 자리는 다른 경우
        - [x]  아웃 : 숫자도, 자리도 다른 경우
    - [x]  입력한 3자리 숫자가 정답과 같은 경우, 게임이 종료됩니다.
        - [x]  ‘3 스트라이크’라면, 정답에 해당합니다.
        - [x]  정답일 때, 표시하고 싶은 메세지는 여러분이 지정해주세요.
    - [x]  올바르지 않은 입력값에 대해서는 오류 문구를 보여주세요.
        - [x]  입력값이 문자, 중복되는 값처럼 요구사항과 맞지 않을 경우
  - [x]  **게임 이어서하기**
    - [x]  정답을 맞출 때까지 계속해서 시도할 수 있어야 하며, 정답을 맞추면 축하 메시지를 출력한다.
  - [x]  **입력값이 유효한지 검사하기**
    - [x]  3자리 수인지 자릿수를 검사합니다.
    - [x]  중복된 숫자가 없는지 중복 숫자를 검사합니다.
    - [x]  입력값에 숫자만 포함되어 있는지 검사합니다.
    - [x]  유효하지 않은 값인 경우 “올바르지 않은 입력값입니다.”를 출력합니다.
  - [x]  **출력 개선**
    - [x]  프로그램을 시작할 때 안내문구를 보여주세요.
        - [x]  1을 입력하면 **“필수 구현 기능”** 의 예시처럼 게임이 진행됩니다.
        - [x]  “2. 게임 기록 보기”는 Lv3에서 제시됩니다. (2는 입력할 수 없습니다.)
        - [x]  3을 입력하면 게임이 종료됩니다.
    - [x]  정답을 맞히는 경우, 안내문구를 다시 보여주세요.
- **Advanced**
  - [x]  **게임 기록 통계**
    - [x]  지금 시도하는 게임이 몇 번째 게임인지 기록합니다.
    - [x]  사용자가 정답을 맞힐 때까지의 “시도 횟수”를 기록하고 게임이 끝났을 때, 총 시도 횟수를 출력합니다.
    - [x]  프로그램 시작 및 정답을 맞히는 경우, 출력 개선에서와 같은 안내 문구를 출력하고, 2를 입력하면 게임의 시도 횟수를 출력합니다.
  - [x]  **출력 개선**
    - [x]  실행 및 정답을 맞힌 경우, 표시되는 안내문구 선택지 개선
        - [x]  3을 입력하면 게임이 종료됩니다.
        - [x]  이전의 게임 기록들도 초기화됩니다.
        - [x]  1, 2, 3 이외의 입력값에 대해서는 오류 메시지 노출
  - [x]  **게임 난이도 조절**
    - [x]  사용자로부터 난이도를 입력받고, 그에 따라 숫자의 자릿수를 조정할 수 있습니다.
    - [x]  자리수는 3, 4, 5자리 숫자 중에서 선택할 수 있습니다.
        - [x]  3, 4, 5 이외의 값에 대해서는 예외처리
    - [x]  자리수를 입력하면 자동으로 게임이 실행됩니다.
